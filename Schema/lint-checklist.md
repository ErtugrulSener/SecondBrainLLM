# Lint-Checkliste

- [ ] Jede Wiki-Notiz besitzt gültiges Frontmatter.
- [ ] Genau ein erlaubtes Typ-Tag ist gesetzt.
- [ ] Alle `sources` liegen unter `Raw/Sources/` und existieren.
- [ ] `source_count` entspricht der Anzahl von `sources`.
- [ ] Jede Raw-Quelle enthält die Pflichtfelder.
- [ ] Keine als verarbeitet markierte Quelle ist ohne Wiki-Abdeckung.
- [ ] Katalog, Manifest und Indizes sind aktuell.
- [ ] Der öffentliche Audit findet keine Geheimnisse oder lokalen Pfade.
