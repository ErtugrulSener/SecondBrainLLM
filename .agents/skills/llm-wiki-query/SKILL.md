# LLM Wiki Query

1. Beginne mit `Wiki/index.md`.
2. Suche mit `search-catalog --query` nach der Nutzerfrage.
3. Öffne nur passende Wiki-Notizen.
4. Öffne deren Raw-Quellen nur bei fehlenden Details oder gewünschter Verifikation.
5. Trenne belegte Erkenntnisse klar von Schlussfolgerungen und nenne die verwendeten Notizen.
